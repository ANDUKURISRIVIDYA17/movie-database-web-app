# Movie Database Web App

A sleek, responsive frontend web application that allows users to search, explore, and view detailed information about their favorite films using a dynamic movie database API.

## 🚀 Features
* Fully responsive layout that adapts gracefully to mobile, tablet, and desktop screens.
* Live search functionality to discover movies instantly.
* Dynamic content rendering without page reloads.
